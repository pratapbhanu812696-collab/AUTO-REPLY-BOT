# Auto Bot Reply

Auto Bot Reply is a web interface designed for transforming customer interaction, powered by Botpress and infused with a custom design. 

## Features
- **Integrated Chatbot Widget:** Powered by Botpress.
- **WhatsApp Integration:** Easily redirect to specific WhatsApp contacts or channels.
- **Sleek Interface:** Designed to impress right out of the box with glassmorphism backdrop filters and custom typography.

## Tech Stack
- HTML5
- Vanilla CSS3
- Botpress SDK

## Usage
Simply launch `index.html` in your favorite web browser. 

*(If you are setting this up further, you might want to use a local web server to prevent cross-origin issues with external scripts or assets).*
